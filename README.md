# nyc-electric-warehouse
Data engineering pipeline for NYC Electric Consumption &amp; Cost dataset (2010–2025)
## Data Sources

Primary dataset:

- **NYC Open Data – Electric Consumption and Cost (2010–May 2025)**  
  Main page: https://data.cityofnewyork.us/Housing-Development/Electric-Consumption-And-Cost-2010-May-2025-/jr24-e7cr  
  CSV download (used by extraction script): https://data.cityofnewyork.us/api/views/jr24-e7cr/rows.csv?accessType=DOWNLOAD  
  Columns / data dictionary (API metadata): https://data.cityofnewyork.us/api/views/jr24-e7cr/columns.json
